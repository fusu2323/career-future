# CLAUDE.md

本文档为 Claude Code (claude.ai/code) 在本代码库中工作提供指导。

## 项目概述

基于 AI 的大学生职业规划智能体，为明杉科技开发。系统通过多维度分析将学生与岗位进行匹配，并生成个性化职业发展报告。

## 技术栈

- **大语言模型**: 通义千问 Qwen-Max 或 智谱 GLM-4（中文优化）
- **后端**: Python + FastAPI
- **前端**: Vue3 + Element Plus + ECharts
- **向量数据库**: Milvus 或 Chroma（用于 RAG 岗位数据检索）
- **知识图谱**: Neo4j（用于职业路径可视化）

## 核心架构（四层）

1. **数据层** — 岗位数据（10000 条）、用户简历、知识图谱存储
2. **算法/模型层** — LLM 集成、RAG 检索链路、匹配算法 (S_total)、图谱寻路
3. **业务逻辑层** — 简历解析 API、图谱查询 API、匹配计算 API、报告生成 API
4. **应用层** — 学生端界面，包含测评问卷、雷达图、职业路径可视化、报告导出

## 核心算法

**多维度人岗匹配** — 4 个加权维度：
- S_base — 基础要求（学历/专业/意向城市）
- S_skill — 专业技能（编程语言/工具）
- S_soft — 职业素养（沟通/抗压）
- S_potential — 发展潜力（学习能力/创新）

公式：S_total = w1·S_base + w2·S_skill + w3·S_soft + w4·S_potential

权重根据岗位类型动态调整（如算法工程师 w2、w4 较高，项目经理 w3 较高）

## 核心模块

1. **岗位画像与知识图谱** — 解析 1 万条岗位数据，构建 10+ 核心岗位画像，可视化垂直/横向职业路径
2. **学生能力画像** — 简历解析、技能提取、雷达图评分
3. **人岗匹配引擎** — 四维加权打分、差距分析（已掌握/需强化/未掌握）
4. **职业报告生成器** — 结构化报告含自我认知、职业探索、目标设定、行动计划；支持 PDF 导出
