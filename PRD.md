# 产品需求文档（PRD）
## 基于 AI 的大学生职业规划智能体

**版本**: 1.0
**日期**: 2026-03-02
**产品方**: 明杉科技

---

## 一、产品概述

### 1.1 产品定位

基于 AI 的大学生职业规划智能体是明杉科技"精准就业平台"的核心引擎，面向高校大学生提供智能化、个性化、可落地的职业规划服务。

### 1.2 目标用户

- **主要用户**: 在校大学生（大一至大四）
- **次要用户**: 高校就业指导中心、辅导员
- **企业方**: 明杉科技合作院校

### 1.3 核心价值

| 竞品类型 | 局限性 | 我们的破局点 |
|---------|--------|-------------|
| 传统测评类（霍兰德/MBTI） | 标签化严重，缺乏真实就业市场连接 | 动态图谱：展示晋升路径（垂直）+ 换岗路径（横向） |
| 招聘平台工具（Boss/智联） | 偏向即时求职，对大学生包容性差 | 多维度量化评估：四维打分模型让差距具象化 |
| 泛用型大模型（ChatGPT 等） | 依赖 Prompt 能力，内容假大空易幻觉 | 行动计划表：倒推至大学阶段具体任务 |

## 五、核心算法

### 5.1 多维度人岗匹配算法

**匹配度计算公式：**

```
S_total = w1 × S_base + w2 × S_skill + w3 × S_soft + w4 × S_potential
```

**四个维度说明：**

| 维度 | 符号 | 权重示例（算法工程师） | 权重示例（项目经理） | 评估内容 |
|-----|------|---------------------|-------------------|---------|
| 基础要求 | S_base | 0.2 | 0.2 | 学历、专业、意向城市 |
| 职业技能 | S_skill | 0.5 | 0.3 | 编程语言、工具框架、证书 |
| 职业素养 | S_soft | 0.1 | 0.4 | 沟通能力、抗压能力、团队协作 |
| 发展潜力 | S_potential | 0.2 | 0.1 | 学习能力、创新参与、项目经历 |

**动态权重机制：**
- 技术类岗位：S_skill 权重最高（0.4-0.5）
- 管理类岗位：S_soft 权重上升（0.3-0.4）
- 研究类岗位：S_potential 权重较高（0.2-0.3）

### 5.2 人岗匹配准确率指标

| 指标 | 目标值 | 验证方式 |
|-----|-------|---------|
| 关键技能匹配准确率 | ≥80% | 抽样比对人工评估结果 |
| 画像关键信息准确率 | ≥90% | 抽样评估岗位/学生画像 |

## 六、API 接口设计（概要）

### 6.1 简历解析 API

```
POST /api/resume/parse
请求：multipart/form-data (简历文件)
响应：{ student_profile: {...}, skills: [...], score: 85 }
```

### 6.2 人岗匹配 API

```
POST /api/matching/calculate
请求：{ student_id, job_id }
响应：{ match_score: 78.5, gap_analysis: {...}, dimensions: {...} }
```

### 6.3 岗位图谱 API

```
GET /api/graph/career-path?job_id=xxx&type=vertical
响应：{ nodes: [...], links: [...] }
```

### 6.4 报告生成 API

```
POST /api/report/generate
请求：{ student_id, target_job_id }
响应：{ report_id, pdf_url }
```
