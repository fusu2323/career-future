<task type="auto">
  <name>Task 2: Unit tests for service layer (retry, timeout, JSON parse retry)</name>
  <read_first>
    - tests/test_llm_service.py (new file)
    - app/services/llm_service.py (from 05-01)
  </read_first>
  <action>
    Create `tests/test_llm_service.py` with the following test functions using `pytest` and `pytest.mark.parametrize`:

    **test_retry_not_on_400**
    - Arrange: mock client raises APIStatusError(400) on every call
    - Act: call `generate_structured("profile", mock_client, "test prompt")`
    - Assert: `LLMRetryExhaustedError` is raised after exactly 1 call (no retries, D-02)
    - Verify via `mock_client.chat.completions.create.call_count == 1`

    **test_retry_not_on_401**
    - Arrange: mock client raises APIStatusError(401) on every call
    - Act/Assert: same as above, call count == 1

    **test_retry_yes_on_500**
    - Arrange: mock client raises APIStatusError(500) then succeeds
    - Act: call `generate_structured("profile", mock_client, "test prompt")`
    - Assert: call count == 2, returns parsed JSON

    **test_retry_yes_on_503**
    - Arrange: mock client raises APIStatusError(503) then succeeds
    - Act/Assert: call count == 2

    **test_backoff_intervals**
    - Arrange: mock client raises APIStatusError(500) 3 times then succeeds
    - Act: call `generate_structured` and measure time between retries
    - Assert: intervals are approximately 1s, 2s, 4s (within 20% tolerance) per D-02

    **test_profile_timeout_15s**
    - Arrange: mock client has 0 delay
    - Assert: `TIMEOUTS["profile"] == 15.0`
    - Act: call with task_type="profile" — verify timeout passed to SDK is 15.0

    **test_report_timeout_45s**
    - Arrange/Assert: `TIMEOUTS["report"] == 45.0`

    **test_match_timeout_20s**
    - Arrange/Assert: `TIMEOUTS["match"] == 20.0`

    **test_json_retry_succeeds_second_attempt**
    - Arrange: first call returns non-JSON string "not json", second returns valid JSON
    - Act: call `generate_structured`
    - Assert: returns valid dict, call count == 2

    **test_json_retry_fails_both**
    - Arrange: mock client returns non-JSON on first 3 calls
    - Act: call `generate_structured`
    - Assert: `LLMJSONParseError` raised, call count == 3 (initial + 2 retries, D-04)

    **test_is_retryable_true_for_timeout**
    - Import `is_retryable_http_error` from llm_service
    - Assert: `is_retryable_http_error(APITimeoutError()) == True`

    **test_is_retryable_true_for_500_503_429**
    - Assert: `is_retryable_http_error(APIStatusError(status_code=500)) == True`
    - Assert: `is_retryable_http_error(APIStatusError(status_code=503)) == True`
    - Assert: `is_retryable_http_error(APIStatusError(status_code=429)) == True`

    **test_is_retryable_false_for_400_401**
    - Assert: `is_retryable_http_error(APIStatusError(status_code=400)) == False`
    - Assert: `is_retryable_http_error(APIStatusError(status_code=401)) == False`

    Use `@pytest.mark.asyncio` for async test functions. Use `pytest.raises` for exception assertions.
  </action>
  <verify>
    <automated>pytest tests/test_llm_service.py -x -v --tb=short 2>&1 | tail -20</automated>
  </verify>
  <done>All 12 unit tests pass. Retry correctly skips 400/401, retries 500/503/429, backoff intervals match 1s/2s/4s, timeouts match D-03, JSON parse retry works.</done>
</task>
