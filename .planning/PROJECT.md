# 基于AI的大学生职业规划智能体

## What This Is

面向大学生的AI职业规划智能体，通过上传简历自动解析学生就业能力画像，对接企业招聘数据构建的岗位画像与关联图谱，生成个性化人岗匹配分析与职业生涯发展报告。帮助学生清晰认知自身能力与目标岗位的差距，制定可操作的行动计划。

## Core Value

帮助大学生从"盲目规划"走向"精准匹配"——通过AI分析让学生清楚知道自己"能做什么"、"缺什么"、"该怎么补"。

## Requirements

### Active

- [ ] **JOB-PROFILE-01**: 构建不少于10个就业岗位画像，每个画像包含专业技能、证书要求、创新能力、学习能力、抗压能力、沟通能力、实习能力等维度
- [ ] **JOB-PROFILE-02**: 建立垂直岗位晋升图谱，呈现岗位晋升路径与关联信息
- [ ] **JOB-PROFILE-03**: 建立换岗路径图谱，将相关岗位血缘关联，规划岗位转换路径，至少5个岗位的换岗路径，每个岗位不少于2条路径
- [ ] **STUDENT-PROFILE-01**: 学生通过简历上传或自行录入方式生成就业能力画像，包含专业技能、证书、创新能力、学习能力、抗压能力、沟通能力、实习能力等维度
- [ ] **STUDENT-PROFILE-02**: 对学生就业能力进行完整度评分和竞争力评分
- [ ] **MATCHING-01**: 人岗匹配度分析，从基础要求、职业技能、职业素养、发展潜力4个维度量化分析
- [ ] **MATCHING-02**: 关键技能匹配准确率不低于80%；岗位画像和学生画像关键信息准确率超过90%
- [ ] **REPORT-01**: 职业目标设定与职业路径规划，清晰呈现发展路径
- [ ] **REPORT-02**: 短期、中期分阶段个性化成长计划，包含学习路径、实践安排
- [ ] **REPORT-03**: 行动计划评估周期与指标，支持动态调整
- [ ] **REPORT-04**: 报告智能润色、内容完整性检查、编辑优化、一键导出

### Out of Scope

- 用户认证与登录系统——当前聚焦核心匹配与报告功能
- 真实招聘平台对接——使用提供的10000条离线数据
- 移动端原生应用——Web端优先
- 多语言支持——中文单语言

## Context

**数据资产**：
- 岗位数据集：9958条记录，12个字段（岗位名称、地址、薪资范围、公司名称、所属行业、公司规模、公司类型、岗位编码、岗位详情、更新日期、公司详情、岗位来源地址）
- 数据来源为计算机类信息化相关行业，涵盖多个发展阶段岗位

**技术积累**：
- 明杉科技已有精准就业平台和人岗匹配算法基础
- 现有后端已实现FastAPI + ChromaDB + Neo4j技术栈
- 已有简历解析、学生能力画像、四维匹配算法代码

**项目性质**：
- 竞赛命题：基于AI的大学生职业规划智能体
- 命题方：明杉科技
- 评分标准：人岗匹配准确率≥80%，画像准确率≥90%

## Constraints

- **LLM**: 必须使用大语言模型作为核心（glm-4等）进行画像生成、匹配、报告生成
- **数据规模**: 10000条岗位数据，约100个岗位
- **匹配维度**: 人岗匹配从基础要求、职业技能、职业素养、发展潜力4个维度分析
- **准确率指标**: 关键技能匹配≥80%，画像准确率≥90%
- **技术栈**: 现有Python后端（FastAPI）+ 前端技术
- **开发周期**: 竞赛项目，周期有限
