# 需要强化

job_required = ["Python"]

# 结果：Python 在 needs_improvement

## 五、核心功能验证

### 5.1 技能差距分析流程
```
1. 提取学生技能列表
   ↓
2. 对比岗位必需技能
   ↓
3.  categorize 到三列表:
   - mastered (已掌握)
   - needs_improvement (需强化)
   - not_learned (未掌握)
   ↓
4. 生成优先行动项 (按优先级排序)
   ↓
5. 生成学习建议 (含预估时长)
   ↓
6. 验证三列表无交集
   ↓
7. 输出完整分析报告
```

### 5.2 API 接口

**POST `/api/matching/gap-analysis`** - 技能差距分析
```json
// 请求
{
    "student_skills": ["Python", "MySQL"],
    "job_required_skills": ["Python", "Redis", "Docker"],
    "job_preferred_skills": ["Kubernetes"],
    "student_proficiency": {"python": "advanced"}
}

// 响应
{
    "mastered": ["Python"],
    "needs_improvement": [],
    "not_learned": ["Redis", "Docker", "Kubernetes"],
    "priority_actions": [...],
    "learning_suggestions": [...],
    "summary": {...}
}
```

**POST `/api/matching/gap-analysis/full`** - 完整差距分析
```json
// 请求
{
    "student_profile": {...},
    "job_profile": {...}
}

// 响应
// 包含完整的差距分析报告
```
